# GoodRecipe 内置菜谱扩充 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 让 App 自带 7 类共 140 条菜谱（含 tags/nutritionalNotes/preparationTips）并迁移 Kotlin 编译配置到 Gradle 9 推荐的 DSL。

**Architecture:** 利用 Python 脚本在 `scripts/generate_seed.py` 中为 6 类动态生成前缀+核心组合，输出到 `res/raw/good_recipe_seed.json`；新增 `SeedDataProvider`/`SeedDataInitializer` 负责读取 JSON 并在 App 启动时填充数据库，确保配合现有 `RecipeRepository`；`app/build.gradle.kts` 里的 Kotlin 编译配置换成 `compilerOptions`。

**Tech Stack:** Python 生成数据脚本，Android/Kotlin + Hilt/Room + Gradle Kotlin DSL。
